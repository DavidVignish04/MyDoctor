# MyDoctor

MyDoctor is a web-based appointment management system that allows patients to book medical appointments and view their booking details in real time.

## Features
- Patient appointment booking
- View booked appointments
- Simple and clean UI using EJS
- Backend handling with Node.js and Express

## Tech Stack
- Node.js
- Express.js
- MongoDB
- EJS
- JavaScript

## Setup Instructions
1. Clone the repository
2. Install dependencies using `npm install`
3. Configure environment variables
4. Run the server using `node src/index.js`
